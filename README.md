# Some-Numerical-Methods-Project
Implementations of common numerical methods in Python, with Gnuplot visualizations and a shell script to run everything.

## Methods Covered

**Root-Finding**
- Bisection Method
- Secant Method
- Newton-Raphson Method
- Fixed-Point Iteration

**Other**
- Euler's Method (ODE solving)
- Naive Gaussian Elimination (matrix decomposition)

## Files

| File | Description |
|------|-------------|
| `methods.py` | Python implementations of all numerical methods |
| `plot.gp` | Gnuplot script for fixed-point and Euler visualizations |
| `run.sh` | Shell script to execute the Python and Gnuplot files |
| `report.pdf` | Written report on the methods and results |

## Usage

```bash
bash run.sh
```

## Requirements
- Python 3
- Gnuplot
